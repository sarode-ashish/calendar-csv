# calendar-csv
#### Prerequisites:-
1) PHP 7.2.*
2) Composer 1.7.2
3) phpunit PHPUnit 6.3.1(less/greater)

#### Steps To Run:-
1) composer install
2) php bin/console  app:create-calendar-csv file_name.csv 2020(Optional)

#### Run Test Case:-
1) phpunit

##### Note:-
    - Files will be created in "/public" directory.
